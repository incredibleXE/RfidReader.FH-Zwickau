# RfidReader.FH-Zwickau
Ein RFID Reader Programm, welches nach UHF Readern sucht und sich darauf verbindet
## Wichtig
Durch die lib/rxtx-rebundled-2.1-7r2-3.jar ist das Tool nur auf Windows lauffähig. Um es auch auf Linux zum laufen zu bringen müsst ihr lediglich die http://java2s.com/Code/Jar/r/Downloadrxtx22pre2jar.htm runterladen, einbinden und die passende rxtxSerial.dll in den JAVA_PATH\lib\i386 kopiert werden
